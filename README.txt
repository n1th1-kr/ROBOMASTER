To add proper icons to this extension, you have two options:

## Option 1: Generate Icons Online
1. Go to: https://www.favicon-generator.org/ or similar
2. Create 16x16, 48x48, and 128x128 pixel versions
3. Save as PNG files:
   - icon-16.png
   - icon-48.png
   - icon-128.png
4. Place them in this directory

## Option 2: Use Placeholder Icons
For now, you can create simple placeholder .png files or use an SVG-to-PNG converter.

The manifest.json already references:
- icons/icon-16.png (16x16)
- icons/icon-48.png (48x48)
- icons/icon-128.png (128x128)

Place the icon files here to complete the setup.
